Critères de Recherche Documentaire

Guide des opérateurs de recherche et exemples d'utilisation pour interroger le catalogue

Cette page présente les différents critères de recherche disponibles pour interroger la base de données documentaire. Chaque critère est associé à un opérateur spécifique et peut être utilisé avec des opérateurs booléens pour affiner les recherches.

Mode d'emploi : Utilisez la syntaxe opérateur:valeur dans le champ de recherche. Par exemple : author_t:philippe besson 1967 pour rechercher les documents de cet auteur.

Le bookmarklet charge le script depuis GitHub Pages — mettez à jour grsci.js sans jamais recréer le bookmarklet.

Comment installer

  1. Affichez la barre de favoris avec Ctrl+Maj+B si elle n’est pas visible
  2. Glissez le bouton 👉 GRSCI™ ci-dessus dans la barre
  3. Rendez-vous sur le catalogue InMedia et cliquez sur le bookmarklet

Comment utiliser

  1. Cliquez sur le bookmarklet depuis le catalogue InMedia
  2. Survolez les titres des documents pour les ajouter à votre sélection
  3. Réordonnez par glisser-déposer dans le panneau gauche NEW
  4. Supprimez individuellement avec × ou en cliquant sur le token NEW
  5. Copiez la requête ou exportez en .txt NEW

⚠ Ne fonctionne pas avec les ressources en ligne. 👉 Aide opérateurs de recherche
